# 🍽️ Restaurant Analysis Dashboard

## 📌 Project Overview

This project presents an interactive data analytics dashboard to analyze restaurant performance, customer preferences, and service factors. It helps uncover insights into ratings, cuisine trends, pricing, and consumer behavior.

## 🎯 Objectives

* Analyze customer satisfaction based on service and food ratings
* Identify popular cuisines and pricing trends
* Understand the impact of factors like alcohol service, smoking, and franchise status
* Discover top-performing restaurants and key business drivers

## 📊 Features

* Dynamic dashboards with filters (City, State, Price, Rating, etc.)
* Visualizations for ratings, cuisine distribution, and customer behavior
* Comparative analysis (Franchise vs Non-Franchise, Alcohol service impact)
* City-wise and demographic insights

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** – Data cleaning and preprocessing
* **SQL** – Data extraction and dataset creation
* **Power BI** – Data visualization and dashboard development
* **Excel/CSV** – Data storage

## 🧹 Data Cleaning (Python)

* Handled missing values and removed duplicates
* Standardized categorical values (e.g., cuisine, city names)
* Converted data types for accurate analysis
* Filtered irrelevant columns and cleaned inconsistent entries

## 📈 Key Insights

* Mexican cuisine is the most preferred among consumers
* Restaurants without alcohol service received higher overall ratings
* Non-franchise restaurants dominate in terms of customer ratings
* Young consumers (18–25) contribute the most to food ratings


## 🚀 How to Use

1. Run the Python notebook for data cleaning
2. Load the cleaned dataset into Power BI
3. Open the dashboard file (.pbix)
4. Use filters and visuals to explore insights

## 📌 Conclusion

This project demonstrates how Python-based data cleaning combined with SQL and Power BI can generate meaningful insights to improve restaurant business strategies.
